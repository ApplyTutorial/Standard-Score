## GRE ##
### GRE  ###
GRE 对于 95% 的同学来说都是语言考试中最头疼的事情，著名的备考书籍《要你命三千单词》。

由于相当部分国际学生的词汇水平不足，导致语文部分不能反映出该生的逻辑能力，所以**取消 GRE 成绩 / 在录取中只参考数学成**成为大多数申请项目趋势。

[GRE 最低录取分名录](www.baidu.com)
